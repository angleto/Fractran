# Fractran

This is a simple implementation of a [Fractran](https://en.wikipedia.org/wiki/FRACTRAN) interpreter in scala

To run the program:
`scala Fractran.scala <number> "<fractions>"`

Example:

```bash
Bash %> scala Fractran.scala 72 "455/33,11/13,1/11,3/7,11/2,1/3"`
```
